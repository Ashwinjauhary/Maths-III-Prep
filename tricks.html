Master these expert shortcuts to solve complex math problems in seconds.

Order & Degree Hack

If a differential equation contains derivatives inside transcendental functions like sin(y'), cos(y'), log(y'), or e^(y'), the Degree is NOT DEFINED. The Order is always the highest derivative present.

Euler's Fast Conversion

Always use eⁱᶿ = cos θ + i sin θ. To find the real part of complex powers like iⁱ, remember it is e^(-π/2). This is a common 1-mark question!

Solenoidal vs Irrotational

Solenoidal: Think "D" for Divergence and "Dot" product (∇·F = 0).
Irrotational: Think "C" for Curl and "Cross" product (∇×F = 0).
Conservative fields are always Irrotational.

Fourier Symmetry

Even Function: bₙ = 0 (only Cosine terms remain).
Odd Function: a₀ = 0, aₙ = 0 (only Sine terms remain).
This saves 90% of calculation time in MCQs!

Integrating Factor Tip

For dy/dx + Py = Q, IF is e^(∫P dx). If P is 1/x, IF is simply x. If P is cot x, IF is sin x.
